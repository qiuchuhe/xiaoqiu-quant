## 策略逻辑

### 核心理念

> "打板的本质是概率游戏。3板确认强度，4板是加速段，溢价空间最大。"

### 五步流程

```
第1步：获取今日涨停板股票池
         ↓
第2步：筛选"连续3天涨停"的股票（3板股）
         ↓
第3步：对第3板做5维度质量评分
         ↓
第4步：输出S/A/B/C/D五级评级
         ↓
第5步：生成第4天操作建议
```

### 评分模型（满分100）

| 维度 | 权重 | 逻辑 |
|------|------|------|
| ⚡ 封板速度 | 25分 | 越早越好。竞价封死（一字板）=25分，尾盘偷鸡=低分 |
| 🔒 封板稳定性 | 20分 | 炸板次数越少越好。0次=20分，炸板≥4次=0分（烂板） |
| 📊 量能健康度 | 20分 | 第3板换手3-8%最佳（缩量加速），>25%死亡换手 |
| 💰 封单强度 | 20分 | 封单/流通市值 ≥5%强封，<0.5%弱封易炸 |
| 🏭 板块梯队 | 15分 | 同板块有其他涨停→板块效应→龙头溢价 |

### 操作建议分级

| 级别 | 分数 | 建议 |
|------|------|------|
| 🔥 S级 | ≥85 | 强烈关注，竞价高开3%内可轻仓博弈，-5%止损 |
| 🟢 A级 | 70-84 | 可以博弈，观察开盘强度，不追高 |
| 🟡 B级 | 55-69 | 谨慎参与，等弱转强信号确认 |
| 🟠 C级 | 40-54 | 不建议，大概率冲高回落 |
| 🔴 D级 | <40 | 坚决避开，烂板/尾盘板/死亡换手 |

## 风险提示

1. **打板是高危操作**，单票仓位不超过总资金20%
2. **-5% 硬止损**，到了就割，不幻想
3. **炸板即卖**，第4板封不住立刻走
4. 连续亏损2次 → **休息3天**，等待情绪回暖
5. **本策略仅供学习研究，不构成投资建议**
