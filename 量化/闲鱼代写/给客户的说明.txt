====================================================
  🔥 3进4连板擒龙策略 —— 使用说明
  买了就能用，不懂电脑也行
====================================================

【第一步】安装Python（只做一次，以后不用再装）

  1. 打开浏览器，访问：https://www.python.org/downloads/
  2. 点黄色大按钮 "Download Python 3.x.x" 下载
  3. 双击下载的文件
  4. ⚠️ 关键：勾选底部的 "Add Python to PATH"
  5. 点 Install Now，等它装完

  如果不会，百度搜"Python安装教程"看视频，3分钟搞定。


【第二步】运行策略（每天收盘后点一下）

  双击 "一键运行.bat"

  第一次运行会自动安装组件（1-2分钟，以后就不用等了）
  然后自动拉数据、分析、出报告。


【第三步】看结果

  报告文件：three_to_four_report.md
  → 右键 → 打开方式 → 记事本

  里面有：
  - 今天有几只3连板
  - 每只的评分（S/A/B/C/D）
  - 该不该买、什么价位买、止损设多少


【常见问题】

Q: 提示"没找到Python"？
A: 第一步没装好，重装一遍，记得勾选"Add Python to PATH"

Q: 运行后没反应？
A: 非交易日（周末/节假日）涨停池是空的，等到交易日收盘后再试

Q: 评分准不准？
A: 策略仅供参考，帮你筛掉烂板。最终买不买自己判断。

Q: 能改成4进5、5进6吗？
A: 可以，联系我（小秋）定制，改参数就行。
