# 📊 策略一：均线多头 + 温和放量

## 选股条件

| 条件 | 参数 |
|------|------|
| 股价 | < 20元 |
| 均线 | MA5 > MA10 > MA20（多头排列） |
| 换手率 | 5% ~ 10% |
| 量比 | 1x ~ 10x |
| 排除 | ST / 创业板(300) / 科创板(688) / 北证(8) |
| 获利盘 | < 80%（同花顺F10人工核对） |

## 买入信号

| 买点 | 名称 | 条件 | 确认时机 |
|------|------|------|----------|
| 🥇 A | 回调MA10不破 | 收盘 ≥ MA10×0.98 且 收阳线 | 收盘确认 → 次日开盘买 |
| 🥈 B | 放量站稳MA5 | 收盘 ≥ MA5 且 量比 ≥ 1.5 | 盘中/收盘均可 → 次日开盘买 |

## 卖出信号

| 优先级 | 信号 | 条件 | 动作 |
|--------|------|------|------|
| 🔴 1 | 硬止损 | 亏损 ≥ 5% | 次日开盘无条件卖 |
| 🟡 2 | 趋势破坏 | MA5 下穿 MA10 | 次日开盘卖 |
| 🟢 3 | 止盈 | 盈利 ≥ 10% | 次日开盘卖 |
| 🔵 4 | 放量滞涨 | 量比>2 且 涨幅<1% | 当日卖 |

## 仓位参数

| 参数 | 值 |
|------|-----|
| 本金 | 3000元 |
| 每笔 | 100股 |
| 最大持仓 | 2只 |

## 用法

```bash
cd D:\AI小秋\策略量化\策略1

python scanner.py              # 盘后全量扫描 → 生成明日候选
python scanner.py --once       # 单次快扫（终端输出）
python scanner.py --live 300   # 盘中监控，每300秒刷新
```
